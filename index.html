CSS Shape Tools

This page collects a series of tools for using CSS Shapes from the Exclusions and Shapes Specification.

Show Shapes Bookmarklet

Clicking this will generate an overlay for shapes in your css. To save the bookmarklet, just drag it up to your bookmarks bar. For an example of what you would see, click below. Note: you must be using a browser that supports shape-inside.
